DEFINITION :

The component is a controller class with a template which mainly deals with a view of the application and logic on the page. It is a bit of code can be used throughout an application. The component knows how to render itself and configure dependency injection.

The component contains two important things; one is view and another one is some logic.

How to create a component?
Method 1 : Manual way
  1. Create a file & named as test.component.ts in test folder
  2. Add Decorator metadata with template and selector
  3. add mandatory class
  4. Import to app.module.ts >>TestComponent
  5. Add in declaration array in app.module.ts >> TestComponent
  6. Path : Angular2Component >> src >> test >> test.component.ts
Method 2 : Angular Cli way
  1. Open cmd -> Go to application DIR
  2. type >> ng g component test
  3. Automatically component get created
  4. component reference is added to the app.module.ts file
  5. Path : Angular2Component >> src >> test >> test.component.ts
